В папке templates хранятся шаблоны html всех страниц, которые подставляются в файл index.html

В папке components хранятся js файлы с классами, соответствующие каждому роуту, в которых прописана основная логика каждой страницы

В папке static хранятся статичные элементы (картинки и шрифты)

Layout (sidebar) хранится в отдельном файле в темплейтах и в него подгружаем остальной контент.
Шаблон состоит из левого сайдбара и блока, в который вставляется основной контент страницы (<div class="content-wrapper">),
а сам layout вставляется в <div id="content"></div>

Все стили, которые являются одинаковыми для разных страниц (в основном это контейнер, кнопки и заголовки), собраны в общем
файле styles. В папке income-expenses находятся стили для страниц Доходы и расходы и с ними связанные. В папке income
стили для страниц доходов, в папке expenses для страниц расходов. В папке templates также созданы отдельные папки
для шаблонов и скриптов этих групп страниц.
